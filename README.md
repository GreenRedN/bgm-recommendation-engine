BGM Recommendation Engine

상황 기반(Environment-aware) 정보와 사용자 입력을 결합해
매장·공간별로 가장 적합한 BGM을 자동 추천하는 경량형 음악 추천 엔진입니다.

본 시스템은 다음 요소를 기반으로 점수를 산정해 최적의 음악을 추천합니다.

▶ 조건 가중치 (인테리어·날씨·계절·시간대·밀집도·목적)

▶ 학습 데이터 (조건별 선택 기록 + 미선택 패널티)

▶ 전역 인기도 (selectionCount)

▶ 신곡 가산점

▶ 다양성 패널티 (반복 재생 방지)

이러한 요소를 조합하여
단순하지만 확장 가능하고, 실제 매장에서 바로 적용 가능한 추천 엔진을 구현했습니다.
